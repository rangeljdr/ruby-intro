#Concatenate Two Arrays

## Competencias a Aprender
- Definir un metodo aue acepte y use argumentos
- Usar expresiones condicionales
- Aplicar metodos de string, array, hash y/o enumerable

## Resumen
Define un metodo `array_concat` el cual tome dos `Arrays` como argumentos y devuelva un solo `Array` producido por concatener los argumentos.

Por ejemplo,

```ruby
array_concat([1,2,3], [4,5,6])   # => [1,2,3,4,5,6]
array_concat([10, 4], ["apple"]) # => [10, 4, "apple"]
```
