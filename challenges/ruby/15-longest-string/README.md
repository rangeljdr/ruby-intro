#Find the Longest String in an Array

## Competencias a Aprender
- Definir un metodo aue acepte y use argumentos
- Usar expresiones condicionales
- Aplicar metodos de string, array, hash y/o enumerable

## Resumen
Escribe un metodo `longest_string` el cual reciba como input un `Array` compuesto de `Strings` y que devuelva el string mas largo dentro del `Array`.


Por ejemplo:

```ruby
# 'zzzzzzz' is 7 characters long
longest_string(['cat', 'zzzzzzz', 'apples'])  # => "zzzzzzz"
```

Si el `Array` esta vacio `longest_string` debe devolver `nil`.
